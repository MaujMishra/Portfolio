Event Detection In Cricket


Detects various events associated with a single ball delivery in a cricket match -

Ball release point identification from the bowler.
Bounce point identification on the pitch.
Hit classification by the batsman.
Shot direction identification.

Future Work -

Future work focusses on implementing models and algorithms to detect more events.
An NLP model will be built which will take input the events detected, and output the commentary of a single delivery.
